### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/emilohlund-git/jemmastables.git
   ```
2. Install NPM packages
   ```sh
   yarn install
   ```
3. Start the application
   ```sh
   yarn dev
   ```
